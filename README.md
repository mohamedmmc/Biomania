## 📖 About

**Biomania** is a Flutter mobile app designed to make biology learning fun and interactive through quizzes, challenges and gamified content.

---

## ✨ Features

- 🧪 Interactive biology quizzes
- 🎮 Gamified learning experience
- 📱 Cross-platform (iOS & Android)
- 🏆 Score tracking & leaderboards

---

## 🛠️ Stack

| Technology | Usage |
|---|---|
| Flutter + Dart | Framework & language |
| GetX | State management |
| Shared Preferences | Local storage |

---

## 🚀 Run locally

```bash
git clone https://github.com/mohamedmmc/Biomania.git
cd Biomania
flutter pub get
flutter run
```
